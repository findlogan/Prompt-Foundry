# Repository Audit Skill

## Purpose

Use this skill to review a repository before making major changes.

## Workflow

1. Inspect the project layout.
2. Identify package managers, build tools, and deployment targets.
3. Summarize risky areas, missing tests, and undocumented setup.
4. Recommend the smallest safe next step.

## Output

- current architecture summary
- top risks
- recommended next action
